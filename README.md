# 機材貸出管理システム (Demo Unit Loan Management)

ローカル環境で動作する機材貸出管理Webアプリケーションです。

## フェーズ0 (初期リリース)
- 管理者作成
- ログイン/ログアウト
- ホーム画面でのカテゴリ表示

## 必要要件
- Python 3.8+
- SQLite3 (Python標準)

## インストール手順

1. 必要なライブラリをインストールします。
   ```bash
   pip install -r requirements.txt
   ```

## 起動方法

1. アプリケーションを起動します。
   ```bash
   streamlit run app.py
   ```

2. ブラウザが立ち上がり、アプリが表示されます。

## 初回セットアップ

1. 初回起動時は「初期セットアップ」画面が表示されます。
2. 管理者の氏名、メールアドレス(ID)、パスワードを入力して作成してください。
3. 作成後、ログイン画面に遷移します。

## 構成
- `app.py`: エントリーポイント
- `src/`: ソースコード
  - `database.py`: データベース操作
  - `auth.py`: 認証ロジック
  - `views/`: 画面コンポーネント
- `data/`: データ保存領域 (自動作成)
  - `app.db`: SQLiteデータベース
